
# Setup
Please check the contents of `.env.sample` and set the environment variables
```
cp .env.sample .env
```
Please reopen with DevContainer.

# Let's Try
Please operate inside the DevContainer

## Show Plan
```
terragrunt run-all plan
```

## Apply
```
terragrunt run-all apply
```
